Laser Turret Aimbot
Overview

The Laser Turret Aimbot is a computer vision-based project that enables real-time hand-controlled laser targeting using OpenCV, Python, Arduino, and PyFirmata. The system tracks the user's hand movements through a webcam and translates them into precise turret angles, allowing a motorized laser turret to follow the user's gestures and accurately aim the laser beam.

This project combines computer vision, mathematical angle calculations, and embedded hardware control to create an interactive aiming system.

Features
1. Real-time hand detection and tracking
2. Gesture-based turret control
3. Precise angle calculation for turret movement
4. Arduino-controlled pan and tilt mechanism
5. Laser beam targeting based on hand position
6. Smooth communication between Python and Arduino using PyFirmata
7. Modular code structure for easy customization and expansion.
